# Security Policy


## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately before disclosing it publicly.

### How to Report

**Email**: **dteslifeline.info@gmail.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any screenshots or proof of concept

### Response Time

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days, including:
- Confirmation of the vulnerability
- Severity assessment
- Timeline for fix
- Coordination for disclosure

### What to Expect

- We will work with you to understand and validate the report
- We will notify you when the issue is fixed
- We will credit you in our security acknowledgments (if desired)
- We will coordinate public disclosure timing

Thank you for helping keep the DTES Lifeline Map safe for everyone!
